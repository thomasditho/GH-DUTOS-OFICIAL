# Manual do Usuário - GH DUTOS

Bem-vindo ao sistema de gestão de ativos GH DUTOS. Este guia ajudará você a operar as principais funções.

## 1. Dashboard
A tela inicial mostra um resumo da saúde da sua planta:
- **Alertas Críticos:** Ativos com status de erro.
- **Manutenções Vencidas:** Baseado na periodicidade configurada.
- **Gráficos:** Distribuição de status e desempenho mensal.

## 2. Gestão de Ativos
- **Novo Ativo:** Clique em "Novo Equipamento" para cadastro manual.
- **Importação:** Use "Importar Excel" para cadastros em massa. Siga o modelo de planilha disponível para evitar erros.
- **Filtros:** Você pode filtrar por Cliente (Pasta), Localização ou Status.

## 3. QR Codes e Etiquetas
- No detalhe de cada equipamento, você encontrará o QR Code único.
- **Impressão Individual:** Gera um PDF de etiqueta única.
- **Impressão em Lote:** Na lista de ativos, selecione vários itens e clique em "Imprimir Selecionados". O sistema organizará 15 etiquetas por página A4 (Grid 3x5).

## 4. Registro de Manutenção
Sempre que realizar um serviço:
1. Escaneie o QR Code ou busque o ativo no sistema.
2. Clique em "Registrar Manutenção".
3. Preencha a descrição e anexe o relatório técnico (PDF) se houver.
4. O status do ativo será atualizado e o histórico ficará disponível para consulta futura.

## 5. Configurações de Impressão
Acesse o menu "Impressão" para:
- Alterar o logotipo dos relatórios.
- Mudar a cor principal dos documentos PDF.
- Ajustar textos de cabeçalho e rodapé.
